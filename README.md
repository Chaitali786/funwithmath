# Fun With Math — Kids Math Game

A small, browser-based text math game for kids built with HTML, CSS and vanilla JavaScript.

Players choose how many rounds to play (1–5) and answer simple arithmetic questions (addition, subtraction, multiplication, division) presented via browser prompts. Correct answers score points; more than half correct wins the game.

## Quick start

1. Open `index.html` in your web browser (double-click or serve it via a static file server).
2. (Optional) Open Developer Tools  Console if you want to see debug logs.
3. Click the "Start Game" button and follow the prompts.
